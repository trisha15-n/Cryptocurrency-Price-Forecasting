# Cryptocurrency-Price-Forecasting

The main objective of the project is to analyse the Bitcoin price trends in USA and India and predict the next day closing price. The model is trained on yfinance data and uses Linear Regression to predict the next day closing price.

The business objective of the project can be to provide a investment strategy optimisation by forcasting near-term price movements. It can also help to analyse the historical performance to understand long term trends.

The data was used to train two models - Linear Regression Model and Random Forest Regressor
